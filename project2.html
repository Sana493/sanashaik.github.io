🩺 When the Most Important Data Disappears

Understanding Diabetes Beyond One Number

1. Project Title

“When the Most Important Data Disappears: Can We Still Predict Diabetes?”

2. Motivation

Diabetes prediction models often rely heavily on glucose — the strongest indicator of diabetes. But real‑world healthcare data is often incomplete. This project explores whether machine learning models can still make meaningful predictions when glucose is removed.

3. Dataset

The Pima Indians Diabetes Dataset contains 768 rows and 8 features:

  • Pregnancies
  • Glucose
  • Blood Pressure
  • Skin Thickness
  • Insulin
  • BMI
  • Diabetes Pedigree Function
  • Age

Target variable: Outcome (0 = no diabetes, 1 = diabetes)

4. Data Preprocessing

  • Replaced unrealistic zeros with median values
  • Standardized numerical features
  • Split data into 80% training, 20% testing
  • Used stratification to preserve class balance

5. Exploratory Data Analysis

Glucose had the strongest correlation with diabetes, followed by BMI and Age.

6. Experiments

Experiment 1 — All Features

Highest accuracy. Glucose dominated feature importance.

Experiment 2 — Remove Glucose

Accuracy dropped. BMI and Age became top predictors.

Experiment 3 — Remove Glucose + BMI + Age

Further accuracy decline. Remaining features still carried weak signals.

Experiment 4 — Weak Features Only

Lowest performance but still above random guessing.

7. Models Used

  • Logistic Regression
  • Decision Tree
  • Random Forest (most robust)

8. Evaluation Metrics

  • Accuracy
  • Confusion Matrix
  • Precision, Recall, F1‑score
  • ROC‑AUC (primary metric)

9. Feature Importance

10. Interpretation & Insights

Models can still function without glucose. Random Forest is the most robust. Performance degrades gradually, not instantly. Even weak features contain hidden patterns.

11. Limitations & Ethics

  • Dataset limited to Pima Indian women
  • May not generalize to broader populations
  • Imputation introduces bias
  • Models are predictive, not diagnostic

12. Conclusion

Even when the most important data disappears, meaningful patterns remain. This highlights the importance of building models that are robust and adaptable to incomplete data.
